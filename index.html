Table Tags

Tag Name Description
<table> Table The table tag is the wrapper element for all HTML tables
<thead> Headings The thead tag represents the block of rows that consist of the column lables in the table
<tbody> Body The tbody tag represents the blocks of rows that consist of a body of data within a table
<tr> Table Rows The tr tag represents a row of cells within a table
<td> Table Data The td tag represents a data cell within a table
<tfoot> Table Footer The tfooter tag represents the set of rows at the bottom of a table. Often showing the sum of the data entered


Table Attributes

Attribute Name Description
<colspan> Column Span Defines how many columns a td element should span
<rowspan> Row Span Defines how many rows a td element should span
